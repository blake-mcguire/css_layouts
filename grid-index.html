GRID

CSS Grid Layout is a powerful layout system in CSS that allows for the creation of complex and responsive web layouts with ease. To understand how CSS Grid works, it can be helpful to compare it to a city grid plan.

The Items inside of the grid can be directly related to the buildings inside of a city, giving a home for the data of the webpage

The Outer Container of the grid can be related to the outer limits of the city and is what holds all elements of the grid itself.
