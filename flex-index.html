CSS {Flexbox}

CSS Flexbox, officially known as the CSS Flexible Box Layout Module, is a one-dimensional layout method for laying out items in rows or columns within a container. It allows for the distribution of space along a single axis, making it easier to design flexible and efficient layouts without having to use floats or positioning. Flexbox is particularly useful for aligning content, distributing space around items, and changing their order visually without altering the HTML.

Flexbox Properties

  • Display
    • selector { display: flex|inline-flex; }

    Display defines an element as a flex container, it can be set to flex, or inline-flex.

  • Flex-Direction
    • selector {flex-direction: row|row-reverse|column-reverse;}

    Specifies the direction of the flex item within the conatiner. Values include:

    • Row
    • Row-reverse
    • column-reverse

  • Flex-Wrap
    • selector {flex-wrap: nowrap|wrap|wrap-reverse;}

    Controls Whether the flex items should wrap onto multiple lines Values include:

    • nowrap
    • wrap
    • wrap-reverse

  • Justify-content
    • selector {justify-content: flex-start|flex-end|center|space-around|space-between|space-evenly;}

    Aligns Flex items along the main axis of the container Values Include:

    • flex-start
    • flex-end
    • center
    • space-around
    • space-between
    • space-evenly

  • Align-Items
    • selector {align-items: flex-start|flex-end|center|space-around|space-between|space-evenly;}

    Aligns flex items along the cross axis. Values include:

    • stretch
    • flex-start
    • flex-end
    • center
    • baseline

  • Align-Content
    • selector {align-content: stretch|flex-start|flex-end|center|space-between|space-around;}

    Aligns Flex lines within the flex container. Values Include:

    • stretch
    • flex-start
    • flex-end
    • center
    • space-between
    • space-around

  • Flex-Flow

    • Shorthand for flex-direction and flex-wrap
